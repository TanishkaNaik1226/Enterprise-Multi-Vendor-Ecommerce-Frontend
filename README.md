## Project Status

### Milestone 1
- Backend APIs and authentication were integrated with the frontend.
- Basic login, registration, and role-based access (Customer, Vendor, Admin) were completed.
- Frontend and backend communication was established.

### Milestone 2 (Current Progress)
- Frontend implementation for Milestone 2 has been pushed to this repository.
- The backend from Milestone 1 is being reused for integration.
- Current work includes:
  - Order Processing & Tracking UI
  - Product display improvements
  - Shop navigation
  - Wishlist and Cart integration
  - Vendor product management (including image upload support)
  - Search and filter improvements
  - Bug fixes and frontend-backend integration

> **Note:** This repository currently contains the Milestone 1 backend and the ongoing Milestone 2 frontend implementation.
>  Some features are still under development and will be completed as backend modules become available.
>
> # Contribution Guidelines

Please do **not** commit directly to the `main` branch.

1. Clone the repository.
2. Create a new branch for your work.
3. Make your changes.
4. Commit and push your branch.
5. Create a Pull Request (or merge after review).
